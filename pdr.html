PDR

Pointers

  • A pointer is a memory address (8 bytes) that holds an address of something.
  • &x is the syntax for the memory address of variable x.
  • int * p; is the syntax for declaring a new pointer p that points to integers.
  • *p is the syntax for dereferencing a pointer, that is, following it to its pointee.
  • Setting a pointer equal to another pointer makes it point to the same location.

Dynamic Memory Allocation

  • Allocate things with the new keyword.
  • Deallocate things by providing the pointer to the delete keyword. This doesn't actually delete the pointer itself, but instead only deallocates the memory location it points to.
  • Unlike Java, C++ doesn't have garbage collection, so if you don't deallocate what you've allocated, you'll open a memory hole.

Visibility

  • Use the syntax friend class x; within a class' header file to allow other class x to access its private fields.

Common C++ Errors

  • Seg Fault: Trying to access memory outside of that which has been allocated.
  • Double Free: Trying to delete something that has already been deleted.
